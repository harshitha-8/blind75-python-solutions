## üéØ Common Patterns

### 1. Hash Map / Hash Set
**Used for O(1) lookups to avoid nested loops**
- Two Sum
- Contains Duplicate
- Group Anagrams

### 2. Two Pointers
**Navigate array from both ends or use fast/slow pointers**
- 3Sum
- Container With Most Water
- Linked List Cycle

### 3. Sliding Window
**Maintain a window that slides through the array**
- Longest Substring Without Repeating Characters
- Maximum Subarray

### 4. Tracking Min/Max
**Keep track of running minimum or maximum**
- Best Time to Buy and Sell Stock
- Maximum Subarray (Kadane's Algorithm)

### 5. Dynamic Programming
**Break down problems into subproblems**
- Climbing Stairs
- Coin Change
- House Robber

### 6. Depth-First Search (DFS)
**Recursively explore nodes**
- Tree traversals
- Graph problems

### 7. Breadth-First Search (BFS)
**Level-by-level exploration**
- Binary Tree Level Order Traversal
- Shortest path problems
